Linux Typer

LinuxTyper is a productivity tool written in C that automates repetitive typing tasks at specified intervals in a selected window. The GUI is built with GTK, allowing it to integrate smoothly with most Linux desktops. Users can input a desired time in seconds and up to five different phrases, then select a window by clicking on it. At the specified time interval Linuxtyper chooses a random phrase and types it into the selected window. While other auto-typers require users to keep the cursor on the selected window, LinuxTyper can run in the background. Even if the selected window is minimized, LinuxTyper will automatically maximize it to execute the routine and then minimize it again. LinuxTyper remembers previously used phrases the next time the app is opened.

Social Media Twitter Bot

The Social Media Translation Bot automatically translates the a public account's Twitter, TikTok, and Instagram posts to the user's preferred language. The bot utilizes the Google Cloud Vision and Google Cloud Translation API for both text and image translations. Whenever a new post is made on one of the supported social media platforms, the bot automatically detects and translates the text and/or images, and posts the translation on Twitter. The bot is deployed usuing Docker and is hosted on a Linux server using AWS EC2.

Ribbon Designer Web App

The Ribbon Designer Web App is an online tool that allows users to easily design ribbons for print. The app provides users with a simple and intuitive interface that enables them to add text and motifs to their ribbon designs, change the font type and size, and preview the design as it would appear once printed.
